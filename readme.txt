Share badges earned on ClassBadges.com on WordPress.

== Description ==

Students who have earned badges on ClassBadges can make some of their badges public and share these public badges  via this plugin.

== Installation ==

1. Unzip archive to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Retrieve your API key from the share folder on Classbadges.com
5. Enter your API key in the Plugin Configuration

== Frequently asked questions ==

= Can teachers share badges using this plugin? =

No, only students at this time can share badges.
